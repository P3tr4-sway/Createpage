# Entrance Module State Model

## Core UI states

- Navigation: default / active
- Feature card: default / hover / pressed / disabled
- Bottom bar: visible / hidden (at content bottom)
- Scroll area: top / middle / bottom

## Interaction notes

- 左侧导航点击后滚动到对应 section
- 内容区滚动到底时底栏隐藏
